# Starter dependencies  
*  Axios
*  React
*  Redux
*  Thunk
*  React Router (v4)
*  Jest
*  ESLint
*  Babel
*  Webpack

## Getting started
1. ```git clone git@github.com:dmarczydlo/react-starter.git```
1. ```yarn install```
1. ```yarn dev```
